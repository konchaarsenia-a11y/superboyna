# ТЗ v8 — Подписки, БП, Опросник, Статистика, месяц→неделя

Утверждено ответами владельца 2026-07-21. **Код не писать без «делай».**

## 0. Валюта и константы (BYN)

| Параметр | По умолчанию | Где менять |
|----------|--------------|------------|
| Сыпучее в ПП | `1` = **100 г** (если заголовок не говорит иное) | лист ПП |
| Коэффициент | **2.3** базовый | на клиента; manager + owner |
| Свет | **11** BYN / подписчик | мини-апп (настройки) |
| Доставка | **6** BYN × N доставок/мес | мини-апп |
| Пакеты | У1 / У2 / У3 / УП4; себест. под заголовками в ПП | подсказка кол-ва + ручная правка |

**Себестоимость** = себест_позиций + свет + 6×N + пакеты  

**Цена (грязные)** = (себест_позиций × коэф) + свет + 6×N + пакеты  

**Прибыль (чистые)** = цена − себестоимость  

**На одну доставку:** месячный объём позиции / N (100 г при N=2 → 50 г).

---

## 1. Месяц → неделя

- Из месяцев (+ Контакты): **все данные клиента** (ник, адрес, телефон, сегмент…).
- **Не** брать состав из ПП/АФК/БП. Состав только из Заказа → бронь на дату.
- До согласия менеджера: **preview** (кто → какой день) и оболочка без чужого состава.
- После «да»: расставить людей по Пн–Пт.
- Доукомплект: нового — добавить; уже есть — не трогать; изменили данные/состав — **заменить** заказ.

---

## 2. Навигация

| Роль | Доступ |
|------|--------|
| **manager** | только **Заказ**, **Просмотр**, **Цена** |
| **owner** | Вкладки: **Заказ**, **Нарезка**, **Курьер**, **Склад**. Подменю Заказа: Просмотр, Цена, **Подписки** (ПП\|АФК\|БП\|Опросник), **Статистика**. (Люди — по необходимости / позже) |

Подписки/Статистика не торчат отдельными табами у менеджера. У owner — ровная поднавигация под Заказом.

---

## 3. Подписки (owner)

### Общее
Счётчик ПП; карточка с составом/N/коэф/пакетами/ценами; пачки на ПП/АФК/БП: →АФК, →ПП, удалить; запись в Sheets двусторонняя.

### ПП
CRUD; добавить ПП (чеклист из предложения); спросить «кто из БП перешёл»; удаление с вопросом; →АФК; правка коэфа.

### АФК
Как ПП на паузе; вернуть в ПП; пачки.

### БП
Составы 1 и 2 (и в Заказе, и в БП); себестоимость авто; партнёр — **бейдж в углу**; этапы:  
ожидает 1-ю доставку → получил 1-ю → ожидает 2-ю анкету → ожидает 2-ю доставку → ожидает 3-ю анкету → в раздумьях (+ перешёл / отказ).  
Заказ с флагом БП → писать в **Опросник** и **БП**.

### Опросник
Дата = **когда отправить** анкету. Пометки только **`БП2`** и **`ПП1`**. UI по дню + edit/delete/перенос + пачки.  
Авто: БП? → номер бесплатной доставки → дата анкеты = доставка **+4** (не почта) или **+6** (почта Евро/Бел).  
Instagram-авто — позже.

---

## 4. Цена на заказе / курьер

- Клиент в ПП → цена подписки **авто**.
- N=1 → каждый раз полная сумма.
- N=2 → на **первой** доставке сумма; на **второй** у курьера возле «доставлено» **всегда** вопрос «оплачено ли?»; если нет — показать сумму к оплате; если да — «оплачено».

---

## 5. Статистика (только owner)

Календарный месяц; **авто** сворачивание при смене месяца в архив; просмотр месяца / полугодия / года.

Обязательно: себестоимость; оборот (грязные); прибыль (чистые); динамика по заказам приложения.  

БП: число БП; перешли в ПП / отказались; деньги на БП; потери на неперешедших; по партнёрам + рейтинг конверсии.  

Дополнительно заложить: средний чек/прибыль; доля ПП vs розница; активные ПП / АФК; воронка этапов БП; снапшот `Статистика_Месяцы`.
